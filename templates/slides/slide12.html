Zero Conditional - General Truth

Structure: If + Present Simple, Present Simple

If you divide by zero, the program crashes.

First Conditional - Real Future

Structure: If + Present Simple, will + verb

If you run the tests, they will pass.

Second Conditional - Hypothetical Present

Structure: If + Past Simple, would + verb

If I had more RAM, the system would run faster.

Third Conditional - Hypothetical Past

Structure: If + Past Perfect, would have + Past Participle

If we had tested the code, we would have found the bug.
