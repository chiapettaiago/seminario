🇧🇷 Como o áudio funciona: O sistema reproduz (Simple Present) palavras quando você clica (Simple Present). Nós implementamos (Simple Past) esta funcionalidade para melhorar a pronúncia.

🇺🇸 How the audio works: The system plays (Simple Present) words when you click (Simple Present). We implemented (Simple Past) this feature to improve pronunciation.

## JavaScript Audio System Code
// This function speaks English words aloud
function speakText(text) {
    // The browser creates a new speech instance
    const utterance = new SpeechSynthesisUtterance(text);
    
    // We set English as the default language
    utterance.lang = 'en-US';
    utterance.rate = 0.8;  // Slower for learning
    
    // The system is playing the pronunciation
    speechSynthesis.speak(utterance);
    
    console.log(`User clicked: ${text}`);
}

// Event listeners handle user interactions
document.querySelectorAll('.vocab-card').forEach(card => {
    card.addEventListener('click', () => {
        // This card was selected by the student
        const word = card.getAttribute('data-word');
        speakText(word);
    });
});
